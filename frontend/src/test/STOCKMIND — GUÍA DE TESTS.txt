REQUISITOS PREVIOS
------------------
- Node.js 20 LTS instalado
- Haber ejecutado "npm install" dentro de la carpeta frontend/

CÓMO EJECUTAR LOS TESTS
------------------------
1. Abrir una terminal
2. Navegar a la carpeta del frontend:

   cd frontend

3. Ejecutar los tests:

   Modo una sola vez:
   npx vitest run

   Modo watch (re-corre al guardar cambios):
   npm test

   Ver cobertura de código:
   npm run test:coverage

  QUÉ SE TESTEA Y POR QUÉ

TECNOLOGÍAS USADAS
------------------
- Vitest       → Framework de testing, compatible con Vite
- React Testing Library → Renderiza componentes React en tests
- jest-dom     → Matchers extra para el DOM (toBeInTheDocument, etc.)
- jsdom        → Simula el navegador en Node.js

NOTA SOBRE MOCKS
----------------
Las llamadas a la API (gateway Node.js) son mockeadas en los tests.
Esto significa que los tests NO requieren que el backend esté corriendo.
Se simula la respuesta del servidor para probar el comportamiento
del frontend de forma aislada.
